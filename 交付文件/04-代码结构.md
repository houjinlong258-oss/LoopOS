# 代码结构

> 源代码不重复复制进 `交付文件/`，请直接把整个项目目录交给同事，这份文件只负责告诉对方该从哪里开始看。

## 前后端说明

LoopOS **没有独立的前端 / 后端目录**，是单一 Python 包 `loopos/`。
AGENTS.md 明确 MVP 无 Web UI：所有交互通过 CLI（Typer/Rich + 标准库 fallback）。

## Runtime Flow（核心循环，来自 README）

```text
Goal -> AmbiguityReport -> GoalSpec -> Context Compiler -> Policy OS
     -> AIL instruction -> Scheduler -> Syscall Router -> Adapter
     -> Observation -> Evaluation -> ProgressDelta -> LoopDecision
     -> Trace / governed Memory or Skill -> continue, approval, or halt
```

Kernel 不变量：
- 每个外部动作都是 syscall，每个 syscall 都过 Policy
- 每个状态转移都被 trace，replay 不重复副作用
- 持久化 memory/skill 写入过治理
- 歧义目标不执行，循环有界
- 触发器只造任务，不直接跑工具
- 高风险 Producer 不能自批
