## 已知环境小坑（非项目缺陷，运行时注意）

1. **Windows `nul` 残留文件**：某些 shell 重定向会留下 0 字节 `nul`。
   - 它是 Windows 保留设备名，`rm`/`del` 删不掉
   - 已在 `.gitignore` 加忽略，`git status` 会自动忽略
   - **mypy 必须用 `mypy loopos tests`，不要用 `mypy .`**（后者扫描仓库根会撞 `nul` 崩溃）
2. **PowerShell profile 的 Anaconda hook 报错**：忽略，用仓库 `.venv` 即可
3. **Pytest 系统 temp 清理失败**：已改用仓库本地 `.pytest-tmp/`
