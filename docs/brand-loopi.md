# LoopOS Brand and Loopi

LoopOS is a terminal-native runtime for governed, replayable agent execution.

- Official name: LoopOS
- Mascot: Loopi / 小环狸
- Slogan: Not another agent. The kernel for running agents.
- Community phrase: Let the loop converge.

## Assets

- `assets/brand/loopos-hero.png`: primary README and project overview image.
- `assets/brand/loopi-kernel.png`: detailed kernel portrait.
- `assets/brand/loopi-mark.png`: compact mascot mark.
- `assets/brand/loopi-sticker-sheet.png`: community and release artwork.

Brand artwork is optional in community-facing output. Security errors, policy denials,
audit logs, JSON output, and machine-readable traces must remain plain and professional.
