# Klein Loopi

Klein Loopi is a brand metaphor, not a runtime dependency.

- The Klein bottle represents a continuous governance boundary.
- Loopi moves freely inside that boundary.
- Trace is the visible trail left by each action.
- Policy OS is the boundary membrane.
- Convergence means each return moves closer to the outcome contract.
- Anti-Bloat review trims unnecessary branches and abstractions.

```text
LoopOS is a Klein-bottle runtime for AI agents:
free movement, governed boundaries, reliable convergence.
```

The metaphor may be used in brand and educational material. Security errors, policy decisions, and
audit reports continue to use direct professional language.
