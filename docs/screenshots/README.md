# Screenshot Guide

Capture terminal examples from `docs/demo-flows.md` at a minimum width of 100 columns. Use the
default dark terminal theme, hide local usernames and absolute paths, and never capture API keys,
environment files, database contents, or user data.

Recommended public captures are the dry-run step stream, ambiguous-goal proposal, L5 policy
explanation, SQLite demo summary, webhook approval lifecycle, and release-readiness report.
