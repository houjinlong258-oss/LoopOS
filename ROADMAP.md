# Roadmap

## Open-Source Alpha

- Deterministic Kernel, AIL, Policy OS L0-L5, syscalls, trace/replay, memory and skill governance.
- Goal negotiation, loop convergence, Data Guard mock flow, local index, compute modes, and registry.
- Mock Provider, multi-model, ChatOps, worktree lease, and independent review contracts.

## Next

- Cross-platform execution hardening and isolated backends.
- Signed plugin metadata, compatibility resolution, and registry revocation.
- Authenticated real provider and ChatOps adapters behind explicit opt-in policy.
- Database connectors only after backup/restore verification receives production-grade testing.
- Release automation, package publication, named maintainers, and public security advisories.
